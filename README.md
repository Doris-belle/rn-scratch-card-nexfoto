# rn-scratch-card

React Native Scratch Card which temporarily hides content from user

## Installation

```sh
npm install rn-scratch-card
```

## Usage

```js
import { RnScratchCardView } from "rn-scratch-card";

// ...

<RnScratchCardView color="tomato" />
```
